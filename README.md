# Nemac DE

Десктопное окружение для Arch Linux (X11).

## Установка

```bash
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/lyrka-meow/testtt/main/tools/installer.sh)"
```

## Управление

```bash
nemac        # обновить, удалить
nemac-dev    # инструменты разработчика (сборка, публикация)
```
